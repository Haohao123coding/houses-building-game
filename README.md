## zh-cn / 简体中文

一个建造房子的游戏，使用 [MIT LICENSE](https://choosealicense.com/licenses/mit)，现在已开发到 Alpha 版本。

游戏中包含难度选择、资源的分配、获取和买卖，以及不同房屋建筑的建造。

预计开发列表：

- 结局
- 资源获取队伍
- 更平衡的游戏花费和奖励
- 变化的商品价格与总价显示
- 金钱的小数部分
- 更多的不同难度下的不同
- 更新日志与版本历史
